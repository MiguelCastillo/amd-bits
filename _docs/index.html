module loader with System.import and CJS dependencies in the browser

bit-imports is built on top of bit-loader which provides you with a flexible and powerful plugin system that let's you fine tune how you process your application assests. For example, you can tap into the power of tools like babel - right in the browser - with no bundling tools.
