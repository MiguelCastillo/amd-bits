Class: Bitloader

Bitloader

new Bitloader()

Facade for System module loader and some extras. This provisions you with functionality for loading modules and process them via plugins. Some relevant information is found [here](https://whatwg.github.io/loader/), but semantics are not quite the same. whatwg/loader was more of a model to stay somewhat on track with the spec's affordances.
